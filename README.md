# Africa GRC Summit 1.0 (2026)

## Overview
Africa’s premier executive forum on governance, risk, compliance, cybersecurity and digital trust in an increasingly complex and interconnected world.

**Theme:** “The Future of GRC: Intelligent, Integrated and Insight-Driven”

**Date:** October 2026
**Venue:** Lagos, Nigeria

## Features
- Premium, modern landing page design
- Mobile responsive layout
- Modern typography (Outfit & Inter)
- Smooth scroll animations
- Glassmorphism effects

## Tech Stack
- HTML5
- Vanilla CSS
- Vanilla JavaScript
- Lucide Icons

## Setup

First, install dependencies:
```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Development

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.
